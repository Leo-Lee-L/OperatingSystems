The Operating System Zoo

Mainframe Operating Systems

The textbook mentions "hundreds of gigabytes of data" as if that were extraordinary! My PC has two 2-TB drives.

These machines process small transactions very quickly, and may have hundreds of disks.

Server Operating Systems

Handle web sites, databases, print services, file services.

Examples:

  • Solaris
  • Linux
  • FreeBSD
  • Windows Server

Multiprocessor Operating Systems

A single, large job can be distributed over multiple processors. Some special software must corrdinate their work and assemble a final "answer."

Personal Computer Operating Systems

The focus is on supporting a single user at a time, running some "productivity" applications, games, or using the Internet. Today, all PC OSs support multiprogramming: my Mac is running 393 processes as I write this.

Examples:

  • Windows 7 / 8 / 10
  • Linux
  • Mac OS X
  • FreeBSD
  • ChromeOS ?

Handheld Computer Operating Systems

Android and iOS dominate here. We have multicore CPUs, lots of memory, cameras and phones, and a gaggle of apps.

Embedded Operating Systems

Important feature: in general, you cannot download an app to your toaster or washing machine. This means fewer worries about memory protection, malicious software, and so on. However, I see no reason to think that downloadable toaster apps aren't coming soon enough.

Examples:

  • Ovens
  • Refrigerators
  • Automobiles
  • Washing machines
  • Exercise bike

Sensor-Node Operating Systems

These mostly live outdoors, into text things like motion, smoke, temperature change, precipitation, and so on. Battery life and tiny amounts of memory are the dominant concerns in this area.

Example: TinyOS

Real-Time Operating Systems

Smart-Card Operating Systems

Chip-and-PIN cards

Sometimes these cards run Java!
